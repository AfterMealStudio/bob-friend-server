= 개요

[[overview-http-verbs]]
== HTTP Methods

본 REST API에서 사용하는 HTTP 동사(verbs)는 가능한한 표준 HTTP와 REST 규약을 따릅니다.

|===
| 동사 | 용례

| `GET`
| 리소스를 가져올 때 사용

| `POST`
| 새 리소스를 만들 때 사용

| `PUT`
| 기존 리소스를 수정할 때 사용

| `PATCH`
| 기존 리소스의 일부를 수정할 때 사용

| `DELETE`
| 기존 리소스를 삭제할 떄 사용
|===

[[overview-http-status-codes]]
== HTTP 상태 코드

|===
| 상태 코드 | 용례

| `200 OK`
| 요청을 성공적으로 처리함

| `400 Bad Request`
| 잘못된 요청을 보낸 경우. 응답 본문에 오류에 대한 정보가 담겨있다

| `401 Unauthorized`
| 사용자가 인증되지 않음

| `403 Forbidden`
| 사용자에게 요청 권한이 없음

| `404 Not Found`
| 요청한 리소스가 없음

| `409 Conflict`
| 요청한 정보가 중복됨

|===

=== 전체 조회

[source]
----
전체조회는 Page 객체를 반환한다. content 필드에 조회 결과가 약속들의 배열 형태([])로 주어진다. 각 약속은 한 건 조회와는 달리 댓글, 대댓글을 포함하지 않고, 댓글의 개수(댓글의 개수 + 대댓글의 개수)만 포함한다. Page객체는 첫 번째 페이지인지를 나타내는 first, 마지막 페이지인지를 나타내는 last 필드를 가진다(마지막 페이지일 경우 last가 true).
----

[source]
----

ex) sort=appointmentTime,asc    약속시간 기준 오름차순
    sort=createdAt,desc         작성시간 기준 내림차순
